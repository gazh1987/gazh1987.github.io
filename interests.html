LeafletJS

Posted 8th November 2015


In my research for my fourth year project I dicovered this great JavaScript library called LeafletJS. Leaflet is "an open-source JavaScript library for mobile-friendly interactive maps". Its very easy to use, I got the map below up and running in under twenty minutes. The map here basically checks your current location using your devices GPS and then displays it on the map. Clicking the blue icon on the map shows you how many meters away from the current point Leaflet thinks you are.

The map here is pretty simple, but Leaflet does have plenty of plugins that can make richer applications. A couple of plugins that I find interesting and that I will definatly be experimenting with are Leaflet Routing Machine that can add routing between two points to a Leaflet map, and also Leaflet.GeoSearch that adds support for geocoding (address lookup) to Leaflet.

Check out Leaflet here. Enjoy!

DANI: Dynamic Artificial Non-Intelligence


This year at Dublin Institute of Technology I finally have the opportunity to study Artificial Intelligence which is something that has interested me for some time now. In second year I created a Chat Bot in C# called DANI, which is short for Dynamic Artificial Non-Intelligence. The user can communicate with DANI by typing on the command line. DANI uses collections to remember words and to build a vocabulary. The user can also make DANI learn things by typing "Learn" followed by what the user wants DANI to learn. Example: typing "Learn Michael Schumacher is a Formula One driver" makes Dani store the string "Michael Schumacher is a Formula One driver". The next time Michal Schumacher or Formula one is referenced in a sentance, DANI is able to output this information on screen.
