DANI: Dynamic Artificial Non-Intelligence


This year at Dublin Institute of Technology I finally have the opportunity to study Artificial Intelligence which is something that has interested me for some time now. In second year I created a Chat Bot in C# called DANI, which is short for Dynamic Artificial Non-Intelligence. The user can communicate with DANI by typing on the command line. DANI uses collections to remember words and to build a vocabulary. The user can also make DANI learn things by typing "Learn" followed by what the user wants DANI to learn. Example: typing "Learn Michael Schumacher is a Formula One driver" makes Dani store the string "Michael Schumacher is a Formula One driver". The next time Michal Schumacher or Formula one is referenced in a sentance, DANI is able to output this information on screen.
